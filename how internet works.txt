Diff between web page web browser website web server
	web page
		A document which can be displayed in a web browser such as Firefox, Google Chrome, Opera, Microsoft Internet Explorer or Edge, or Apple's Safari.
		These are also often called just "pages."
		
	website
			A collection of web pages which are grouped together and usually connected together in various ways. Often called a "web site" or simply a "site."
web server
			A computer that hosts a website on the Internet.
search engine
			A web service that helps you find other web pages, such as Google, Bing, Yahoo, or DuckDuckGo. 
			Search engines are normally accessed through a web browser
			(e.g. you can perform search engine searches directly in the address bar of Firefox, Chrome, etc.) or
